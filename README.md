# Vertical-Slice Assinment

## Project Structure

### Frontend (`/frontend`)

**Tools & Technologies:**

- **Next.js 16**
- **React 19**
- **TypeScript**
- **Tailwind CSS**
- **PostgreSQL (pg)**
- **Bun**

### Scraper & Database Seeder (`/scraper-and-db_seed`)

**Tools & Technologies:**

- **Node.js**
- **Puppeteer**
- **Neon Database (@neondatabase/serverless)**

## Usage

### Frontend

```bash
cd frontend
npm install
node index.js // for scraping
node seed.js  // for seeding the database
```

### Scraper & Database Seed

```bash
cd scraper-and-db_seed
npm install
npm start
```
